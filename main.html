Composing a Valid HTML Document

This week, I learned how to structure a valid HTML document using essential tags like <!DOCTYPE>, <html>, <head>, and <body>. At my ALC Taibah Inovation Hub, Bauchi, I was introduce to CSS which stands for Cascading Style Sheet, I used it to centralised my pages. I realised the importance of these elements in creating a functional and accessible webpage. The clear separation between the <head> for metadata and the <body> for content provided a better understanding of web development fundamentals.

Understanding Elements, Tags, and Attributes

A key takeaway from this week was understanding the roles of elements, tags, and attributes in structuring web content. HTML elements serve as the building blocks, while tags bring them to life, and attributes control their behaviour and appearance.

  • Elements give structure
  • Tags activate the elements
  • Attributes enhance functionality and accessibility

Exploring HTML Elements and Attributes

I practised using at least five different HTML elements, including:

  • Headings: h1 through h6
  • Paragraphs: p
  • Lists: ol and ul
  • Links: a
  • Images: img

I also applied attributes such as href, src, alt, and target to control how content is displayed. Accessibility was a key focus as I realised the importance of the alt attribute in making images accessible to all users.

In conclusion, this week equipped me with the foundational skills to create structured and accessible web pages. I'm excited for the next chapter of my learning journey!
